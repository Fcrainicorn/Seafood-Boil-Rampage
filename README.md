# SEAFOOD BOIL RAMPAGE 🦀
A Pygame arcade shooter inspired by Space Invaders, but set in the high-stakes world of last-minute grocery shopping. The store is closing soon, and you still need the ingredients for your seafood boil. Instead of aliens, you are dodging aggressive shoppers to clear the aisles of eggs, corn, shrimp, and crab. And the only way to get your ingredients in time is to blast through them.

## Features
- Classic fixed-shooter arcade gameplay
- Multiple difficulty modes: **WEEKDAY**, **WEEKEND**, and **HOLIDAY**
- Random lobster spawns that unlock the special “Gold Standard” ending
- Custom story with multiple endings
- Unique score-to-currency system
- Generates a `receipt.txt` file for successful runs that also include at least one lobster

## Installation
1. Install Pygame:
   ```bash
   pip install pygame
2. Download or clone the project
3. Run the game:
   ```
   python seafood_boil_rampage.py
   
## Controls
Arrow Keys: Navigate your shopping cart left and right
Spacebar: Fire your Old Bay seasoning (bullets)
Enter: Select difficulty and return your cart after the game ends

## Gold Receipt
A receipt.txt file is generated only if you successfully clear the game and collect at least one lobster.
The receipt contains your final score and itemized results from your run.
